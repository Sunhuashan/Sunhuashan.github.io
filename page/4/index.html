【Linux命令】Linux-02-目录结构

发表于 2022-09-15

在Linux系统中，一切皆是文件

  • 文件目录结构

  • /

    根目录

  • /boot

    存放系统启动时的核心文件

  • /bin

    Binary的缩写，存放经常使用的命令

  • /etc

    存放系统和应用程序的配置文件

  • /sbin

    System Binary 的缩写，存放可对系统进行配置的命令

  • /home

    存放普通用户的主目录，每个用户有一个目录，一般以用户名命名

  • /lib

    存放最基础的动态链接共享库

  • /usr

    用户安装程序的默认目录，类似win下的“Program Files”

  • /usr/local

    应用程序的安装目录

  • /root

    系统管理员的用户主目录

  • /proc 、 /srv 、/sys

    系统相关的文件[勿动]

  • /tmp

    存放临时文件

  • /dev

    将所有的硬件用文件的形式存储起来，类似win下的设备管理器

  • /mnt

    为了让用户挂载外部存储空间，如虚拟机与母机直接的共享存储空间

  • /opt

    应用程序安装包存放的目录

  • /var

    存放着在不断扩充的东西，如日志文件

【工具】搜索引擎及其使用技巧

发表于 2022-09-15 分类于 杂项

baidu.com

  • 减除无关资料

    如: “武侠小说 -古龙”；不含“古龙”信息的“武侠小说”

  • 并行搜索

    如:”图片 | 写真”；搜索“图片”或“写真”

  • 搜索范围限制在网页标题中

    如：”写真 intitle:林青霞“；

  • 搜索范围限制在特定网站中

    如：”写真 site:skycn.com”;

  • 搜索范围限制在URL链接中

​ 如：“photoshop inurl:jijiao”;搜索的url必须包含”jijiao“

  • 精确匹配

    如：“ “上海科技大学” “；双引号不拆分关键词

    如：”《书籍名》“；书名号精确匹配书籍

google.com

  • 按链接搜索

    如：“bit link:google.com”;找出所有指向“google.com”的包含关键词”bit”的网页

  • intitle、inurl、intext
